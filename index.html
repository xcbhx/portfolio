Make My Test Count

The Opportunity Project facilitates 12-week product development cycles where tech teams collaborate to create data-driven solutions for pressing public issues. Specifically, we focus on addressing the data challenges posed by modern in-vitro diagnostic tests, like SARS-CoV-2, which have moved beyond traditional laboratory settings. We encourage teams to develop tools that seamlessly capture and transmit test data while upholding best practice data standards, leveraging emerging technologies like image capture and loT.
